# Export & Reporting — v1.3.0

> Export tracked app data, comparisons, and change reports.

## Current State

- **Backend:** Missing
- **Frontend:** Missing

## Tasks

### 1. Backend — Export Endpoints

#### 1.1 App Export

```
GET /apps/export?format=csv|json
```

Current data for all tracked apps (name, platform, rating, version, category, publisher).

#### 1.2 Changes Export

```
GET /changes/apps/export?format=csv|json&from=2026-01-01&to=2026-04-16
```

Changes within a date range.

#### 1.3 Competitor Comparison Export

```
GET /apps/{platform}/{externalId}/competitors/export?format=csv|json
```

Side-by-side comparison table of app and its competitors.

### 2. Frontend

- "Export" button on every list page (CSV / JSON)
- "Download Report" button on app detail page
- Date range picker (for changes export)

### 3. Implementation Notes

- Use Laravel `StreamedResponse` for CSV (handles large datasets)
- JSON export is direct API response
- No need to queue — generate on the fly
