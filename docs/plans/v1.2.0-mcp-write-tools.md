# MCP Write Tools — v1.2.0

> Add write operations to the MCP server, enabling app tracking and competitor management directly from Claude Code.

## Current State

- **MCP Server:** Published (`@appstorecat/mcp@1.0.1`) — read-only tools only
- **Backend:** Write endpoints already exist (track, untrack, competitor add/remove, publisher import)

## Tasks

### 1. New MCP Tools

**File:** `mcp/src/tools/actions.ts`

| Tool | Method | Endpoint | Description |
|------|--------|----------|-------------|
| `track_app` | POST | `/apps/{p}/{id}/track` | Add app to tracking list |
| `untrack_app` | DELETE | `/apps/{p}/{id}/track` | Remove from tracking |
| `add_competitor` | POST | `/apps/{p}/{id}/competitors` | Add competitor |
| `remove_competitor` | DELETE | `/apps/{p}/{id}/competitors/{c}` | Remove competitor |
| `add_app` | POST | `/apps` | Add new app |
| `import_publisher_apps` | POST | `/publishers/{p}/{id}/import` | Import publisher's apps |

### 2. API Client Update

**File:** `mcp/src/client.ts`

- Add `apiPost(path, body)` function
- Add `apiDelete(path)` function

### 3. Annotations

- Write tools use `readOnlyHint: false`
- `destructiveHint: true` for untrack, remove_competitor

### 4. Register & Build

- `mcp/src/register.ts` → add `registerActionTools`
- `npm run build` → `npm publish`

### Verification

```bash
cd mcp && npm run build
# In Claude Code: "track Instagram" → track_app works
# "add ChatGPT as competitor" → add_competitor works
```
